وحدة النافذة – الوحدات الرئيسية

الوحدة الرئيسية الوصف والوظيفة الأساسية أصحاب المصلحة الرئيسيون المتطلبات الوظيفية الملاحظات
إدارة شكاوى المواطنين استقبال شكاوى المواطنين الواردة من مختلف القنوات (البوابة الإلكترونية، النافذة، مجلس المدينة)، وتصنيفها، وإحالتها إلى الأقسام المختصة، ومتابعة حلها، وإشعار المواطن عند تغيير حالة الشكوى. - موظف النافذة.
- رؤساء الأقسام.
- مجلس المدينة (البلدية).
- المواطنون.		 1. استقبال شكاوى المواطنين إلكترونياً.
2. تصنيف الشكاوى حسب النوع (صيانة، حفر، خدمة، إلخ).
3. إحالة الشكوى آلياً إلى القسم المختص.
4. متابعة حالة الشكوى (جديد/قيد المعالجة/منجز).
5. إشعار المواطن عند تغيير حالة الشكوى.
6. تقارير عن الشكاوى وأوقات الحل.		 - تتكامل مع وحدة المعاملات والمهام.
- تعتبر قناة تواصل مباشرة مع المواطن.
إدارة استقبال الأسئلة والاستفسارات استقبال استفسارات وأسئلة المواطنين عبر قنوات متعددة (مكالمات هاتفية، واتساب، أو عبر النافذة على أرض الواقع)، وتسجيلها وتصنيفها، وإحالتها إلى الجهة المختصة للإجابة، وتوثيق الردود. - موظف النافذة.
- رؤساء الأقسام.
- المواطنون.		 1. استقبال أسئلة واستفسارات المواطنين عبر الهاتف، واتساب، أو على أرض الواقع.
2. تسجيل السؤال أو الاستفسار برقم وتاريخ تلقائي.
3. تصنيف الاستفسار حسب الموضوع.
4. إحالة الاستفسار إلى القسم المختص للإجابة.
5. توثيق الرد وإشعار المواطن بالجواب.
6. تقارير عن الاستفسارات وأوقات الرد.		 - تعتبر قناة تواصل سريعة مع المواطن.
- تسهل متابعة الاستفسارات وضمان الرد عليها.
إدارة الإيميلات الواردة استقبال الإيميلات الواردة من الجهات الخارجية إلى البريد الإلكتروني الرسمي للمديرية، وتسجيلها، وتصنيفها، وإحالتها إلى الجهة المختصة داخل المديرية، مع توثيق الردود. - موظف النافذة.
- رؤساء الأقسام.
- مجلس المدينة (البلدية).		 1. استقبال الإيميلات الواردة إلى البريد الرسمي للمديرية.
2. تسجيل الإيميل برقم وتاريخ تلقائي.
3. تصنيف الإيميل حسب الموضوع والنوع.
4. إحالة الإيميل إلى القسم المختص للمعالجة.
5. رفع الإيميل إلى الجهة المعنية داخل المديرية.
6. توثيق الرد وإرساله إلكترونياً.
7. أرشفة الإيميلات وحفظها رقمياً.		 - تتكامل مع الأرشفة والبحث الرقمي.
- تسهل إدارة البريد الإلكتروني الرسمي.
الأرشفة والبحث الرقمي أرشفة جميع الكتب والمراسلات والشكاوى والاستفسارات والإيميلات رقمياً، مع إمكانية البحث السريع والاسترجاع حسب الرقم أو التاريخ أو الموضوع أو الجهة. - موظف النافذة.
- جميع مستخدمي النظام.		 1. أرشفة رقمية لجميع الكتب والمراسلات والشكاوى والاستفسارات والإيميلات الورقية والإلكترونية.
2. إمكانية البحث بالرقم أو التاريخ أو الموضوع أو الجهة.
3. تصنيف الملفات حسب الفترة الزمنية والنوع.
4. استرجاع الملف مع صورته الأصلية (PDF/JPG).
5. تحديد صلاحيات الوصول للملفات المؤرشفة حسب الدور الوظيفي.		 - تسهل عملية الاسترجاع وتقلل الجهد والوقت.
- تحل محل الأرشفة الورقية التقليدية.
